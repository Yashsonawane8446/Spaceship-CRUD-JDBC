# Spaceship Management System

## Project Overview
The Spaceship Management System is a console-based CRUD application developed using Java, JDBC, PostgreSQL, and Maven. The project demonstrates database connectivity and management of spaceship records through Create, Read, Update, and Delete operations.

## Technologies Used
- Java
- JDBC
- PostgreSQL
- Maven
- Eclipse IDE

## Features
- Insert Spaceship Records
- Update Existing Records
- Delete Records
- View All Records
- Database Connectivity using JDBC
- Exception Handling

## Database Fields
- Ship ID
- Ship Name
- Fuel Level
- Crew Count
- Mission Status
- Launch Date
- Destination

## How to Run
1. Create the PostgreSQL database.
2. Update database credentials in the project.
3. Run `SpaceshipDriver.java`.
4. Choose operations from the menu.
